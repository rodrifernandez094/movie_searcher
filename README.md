# movie_searcher

## Getting Started

First, run the development server:

```bash
*npm run dev
*cd client
*npm run start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Learn More
Web app to Find info about millions of movies. Powered by TMDB api.
